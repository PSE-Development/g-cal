G-cal combines a simple calendar with gamification features like achievements, experience points, reports and friends.
It allows you to create ppointments and to-dos, share events with friends, and group them together in groups,

The project consists of a client and server module, both written in Kotlin. For deployment of the server, see installation guide.

G-cal was developed for the 'Praxis der Softwareentwicklung' module during WS25/25 at KIT.
The waterfall model was used for the development of the application, the documentation can be found under /documentation.
